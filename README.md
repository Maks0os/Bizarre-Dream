# Bizarre-Dream
игра с видом сверху
Управление
WASD/Стрелки - движение
F - включить/выключить фонарик
E - взаимодействие
Мышь - направление фонарика
Технологии
Python + Pygame - основной движок
Tiled - создание карт (.tmx)

Как запускать:
необходимые библиотеки:
pygame
pytmx

открыть файл game.py в любом ide python
